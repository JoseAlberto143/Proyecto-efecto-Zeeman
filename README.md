# Proyecto-efecto-Zeeman

——Simulacion del efecto Zeeman en atomos topo alcalino——

La etapa de calibracion se da por el archivo run_pipeline.py

Para correr todas las especies: python3 run_pipeline.py --all

Genera: Out: gráficas de error, algunos potenciales radiales y un csv con los datos de los átomos analizados

La etapa de Zeeman se da por el archivo run_zeeman.py

Para correr todas las especies: python3 run_zeeman.py --all

Genera: Out_Zeeman: gráficas del desdoblamiento Zeeman por especie

Librerías necesarias:
Numpy,
Scipy,
Matplotlib,
Sympy,
Pandas,
Requests y 
Numba
